Character Animation

The following are character animation works I've created. In some instances the rigs were given to me but others I've rigged myself.

The Easter Bunny (2017)

A lot of hard work and development went into this character. Initially it was created for a rigging class, the model was given and our assignment was to make a comprehensive rig and do a 30 second test animation. The second I saw the model, the plotline struck me and I created The Easter Bunny. I created the baseline script and used a friend for the voice acting.

Sugar Scope Cereal (2015)

This was created to experiment with different aspects of the program Cinema 4D. I wanted to capture the essence of the imagination of a child in this commercial for a magical cereal.

Dinosaur Rig & Animation (2017)

Animation and rig done by myself. This was just a short animation to test out how my rig was working in C4D.

Chipmunk (2017)

Animation and rigging done by myself. Dynamics applied to shirt. Created for use in a game featured currently on Boston Children's Hospital's interactive wall. In the game the character peeks and pops through a door as it is triggered by passerby's movement.

Twiggy (2017)

Animation and rigging done by myself. Created for use in a game featured currently on Boston Children's Hospital's interactive wall. In the game the character peeks and pops through a door as it is triggered by passerby's movement.
